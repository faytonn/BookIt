# Features Directory Structure

This directory contains the main features of the Event Reservation System:

## Authentication
- User registration
- Login
- Password reset
- Email verification

## Events
- Event listing
- Event details
- Event creation (for organizers)
- Event management
- Search and filtering
- Waitlist management

## Reservations
- Ticket booking
- Seat selection
- Payment processing
- Reservation management
- Email notifications

## Admin
- User management
- Event oversight
- Reports and analytics
- Content management
- System settings

## Profile
- User profile management
- Booking history
- Saved events
- Notifications settings
